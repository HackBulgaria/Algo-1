# Roots

Implement a function in [Java](Roots.java), [Python](roots.py) or [C++](roots.cpp)
that computes the square root of a number using binary search with precision of `0.00001`.

## Example 1

Input:
```
5
```

Output:
```
2.23607
```

## Example 2

Input:
```
4
```

Output:
```
2.00000
```

## Example 3

Input:
```
247359
```

Output:
```
497.35199
```
